По почасовке:
Шаг первый, достаем список согильдийцев
https://www.bungie.net/Platform/GroupV2/{clanId}/Members/
Шаг второй,  в списке берешь Id профиля, достаешь для каждого id персонажей
https://www.bungie.net/Platform/Destiny2/4/Profile/{profileId}/?components=Profiles
Для каждого персонажа достаешь список активностей:
https://www.bungie.net/Platform/Destiny2/4/Account/{profileId}/Character/{сharacterId}/Stats/Activities/?count=250&page=0
тебе придет массив из 250 объектов. в каждом есть дата мероприятия:
Response.activities[0].period
и длительность:
Response.activities[0].values.activityDurationSeconds.basic.values
Надо чекать даты и складывать длительность. Если эти 250 событий заняли меньше недели, а ты хочешь узнать стату за неделю, достаешь вторую страницу:
https://www.bungie.net/Platform/Destiny2/4/Account/{profileId}/Character/{сharacterId}/Stats/Activities/?count=250&page=1
И так далее. 
По итогу надо просуммировать значение для всех мероприятий со всех трех персонажей. И получишь время игры за неделю в секундах.
